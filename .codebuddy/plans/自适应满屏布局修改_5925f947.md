## 用户需求

用户希望用例生成页面能够自适应屏幕宽度，实现满屏显示效果。

## 核心问题

- 当前 `style.css` 中的默认模板样式限制了页面宽度
- `#app` 设置了 `max-width: 1280px`，导致内容无法填满屏幕
- `body` 设置了 `display: flex; place-items: center;`，导致内容居中而非全屏

## 期望效果

- 页面内容自适应屏幕宽度，填满整个可用区域
- 配合 Layout 组件的固定侧边栏和顶部导航，实现完整的全屏布局

## 技术方案

### 问题定位

**文件**: `/Users/qiutian/cursor_code/qaitest/frontend/src/style.css`

**问题1** - body 样式 (第25-31行):

```css
body {
  display: flex;
  place-items: center;  /* 导致内容居中 */
}
```

**问题2** - #app 样式 (第61-66行):

```css
#app {
  max-width: 1280px;    /* 限制了最大宽度 */
  padding: 2rem;        /* 添加了内边距 */
  text-align: center;
}
```

### 修改策略

1. **修改 body 样式**: 移除 flex 布局和居中对齐，改为简洁的全屏基础样式
2. **修改 #app 样式**: 移除宽度限制和内边距，改为 100% 宽高
3. **添加 html 样式**: 确保完整的高度链 (html → body → #app)

### 技术细节

- 保留 `margin: 0` 和 `min-height: 100vh` 等有用样式
- 不影响 Layout.vue 组件现有的固定定位布局逻辑
- 兼容 Tailwind CSS 的全局样式 (styles/index.css)
