## 产品概述

对QA测试管理系统的所有详情页进行统一的UI优化设计，提升页面视觉层次感和专业度。

## 核心需求

- **优化范围**：测试用例详情页、功能点详情页、测试计划详情页、测试报告详情页、版本详情页，共5个页面
- **优化重点**：布局和排版 - 优化信息展示的层次感和结构
- **设计原则**：统一风格、现代简约、层次分明、重点突出

## 具体问题

1. 布局单调 - 使用简单的grid布局，缺乏视觉层次
2. 信息展示平淡 - 重点信息不突出，没有主次之分
3. 缺乏装饰元素 - 没有图标、卡片阴影、分隔线等视觉元素
4. 整体风格陈旧 - 缺少现代设计感（渐变、圆角、阴影、间距节奏）

## 技术栈

- **前端框架**：Vue 3 + TypeScript
- **样式方案**：Tailwind CSS
- **组件库**：Element Plus（部分组件）、lucide-vue-next（图标）
- **现有设计系统**：primary色 #409EFF，已定义card类、btn系列、颜色变量

## 实现方案

### 统一详情页设计规范

创建一套统一的详情页布局组件和样式规范：

1. **页面布局结构**

- 顶部导航区：返回按钮 + 面包屑导航
- 头部信息区：主标题 + 状态标签 + 操作按钮组
- 快捷统计区：关键指标卡片（如测试计划/报告页）
- 内容分组区：信息卡片分组展示

2. **卡片设计规范**

- 圆角：rounded-xl (12px)
- 阴影：shadow-sm + hover:shadow-md
- 内边距：p-6
- 边框：border border-gray-100
- 分隔线：border-t border-gray-50

3. **信息展示规范**

- 标签：text-sm text-text-secondary font-medium
- 内容：text-text-primary
- 重点字段：使用图标 + 渐变背景卡片
- 状态标签：统一的颜色体系

4. **测试步骤特殊设计**

- 步骤编号圆形徽章
- 左侧连接线形成流程感
- 预期结果缩进展示

## 设计风格

采用**现代简约商务风格**，强调信息层次感和专业度，适合企业级测试管理系统。

## 设计规范

### 1. 卡片设计

- 圆角：12px (rounded-xl)
- 阴影：0 1px 3px rgba(0,0,0,0.05)，hover时 0 4px 12px rgba(0,0,0,0.08)
- 背景：#FFFFFF
- 边框：1px solid #F0F2F5

### 3. 测试步骤特殊设计

- 步骤编号：圆形背景，渐变色（primary色系）
- 左侧连接线：形成流程感
- 预期结果：缩进 + 浅色背景 + 左边框强调

### 4. 统计卡片设计

- 渐变背景：从浅到深的主题色
- 图标：使用 lucide 图标
- 数字：大字号加粗
- 说明文字：小字号灰色
