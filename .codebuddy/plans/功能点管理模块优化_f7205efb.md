## 用户需求

优化功能点管理模块，实现完整的列表管理模式，支持编辑、查看详情等功能。

## 产品概述

功能点管理是 QAITest 智测平台的核心模块，用于管理从需求文档中提取的功能点。当前模块已有基础列表展示，但缺少搜索、分页、新建、编辑、删除、详情查看等核心功能。

## 核心功能

1. **列表页优化**

- 添加搜索框（按名称/关键词搜索）
- 添加项目/类别/优先级筛选器
- 添加分页器组件
- 添加新建功能点对话框
- 添加编辑/删除操作按钮
- 添加详情跳转入口

2. **详情页新增**

- 显示功能点完整信息
- 支持编辑和保存
- 显示关联的测试用例

3. **后端优化**

- 添加 updated_at 字段
- 添加排序参数支持

## 技术栈

- **后端**: FastAPI + Tortoise ORM + Pydantic
- **前端**: Vue 3 + TypeScript + Element Plus + Tailwind CSS
- **状态管理**: Pinia

## 实现方案

### 1. 后端优化

**添加 updated_at 字段**：在 Requirement 模型中添加更新时间字段

**添加排序参数**：列表 API 支持 sort_by 和 order 参数

### 2. 前端列表页改造

**搜索筛选区**：

- 项目选择器
- 关键词搜索框
- 类别/优先级筛选

**数据表格**：

- 使用 Element Plus Table 组件
- 支持排序
- 操作列：查看/编辑/删除

**分页器**：

- Element Plus Pagination 组件

**新建/编辑对话框**：

- 使用 Element Plus Dialog + Form

### 3. 详情页新增

**路由配置**：`/requirements/:id`

**页面布局**：

- 基本信息卡片
- 关联测试用例列表
- 操作按钮：编辑/删除
