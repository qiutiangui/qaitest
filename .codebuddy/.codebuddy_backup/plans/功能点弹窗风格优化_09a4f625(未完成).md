## 产品概述

优化功能点管理页面的弹窗风格，使其与项目中其他页面（ProjectList、VersionList）保持一致

## 核心功能

- 新建/编辑弹窗：改为 Element Plus 的 el-dialog + el-form 风格
- 详情弹窗：使用 el-descriptions 展示信息，添加编辑按钮
- 保留双入口：列表编辑按钮 + 详情弹窗编辑按钮

## 交互逻辑

- 列表"编辑"按钮 → 直接打开编辑弹窗
- 列表"查看"按钮 → 打开详情弹窗 → 点击"编辑"按钮 → 打开编辑弹窗

## 技术栈

- Vue 3 + TypeScript
- Element Plus 组件库（el-dialog, el-form, el-descriptions）
- Tailwind CSS

### 具体修改

**1. 新建/编辑弹窗改造**

- 使用 `el-dialog` 替代自定义弹窗
- 使用 `el-form` + `el-form-item` 组织表单
- 使用 `el-input`、`el-select` 替代原生输入框
- 宽度设为 600px（字段较多）
- 使用 `#footer` 插槽放置操作按钮

**2. 详情弹窗改造**

- 改用 `el-dialog` 包裹
- 使用 `el-descriptions` 展示字段信息
- 添加底部操作栏：编辑按钮 + 关闭按钮
- 编辑按钮点击后打开编辑弹窗

**3. 交互逻辑调整**

- 详情弹窗的编辑按钮调用 `handleEdit(detailRequirement)` 打开编辑弹窗
- 保持列表编辑按钮原有逻辑
