## 产品概述

用户反馈用例生成界面卡片整体宽度太窄，需要调整两个相关页面（AICaseGeneration/Generate.vue 和 TestCaseGenerate.vue）的卡片尺寸。

## 核心需求

- 调整 AI用例生成页面 (AICaseGeneration/Generate.vue) 的卡片宽度
- 调整 用例生成与管理页面 (TestCaseGenerate.vue) 的卡片宽度
- 增加卡片内边距，使内容更加宽松舒适
- 保持 UI 风格一致性

## 技术栈

- Vue 3 + TypeScript
- Tailwind CSS 样式框架
- Element Plus UI 组件库

## 实现方案

### 问题分析

当前卡片样式定义在 `styles/index.css` 中，使用 `p-4`（16px 内边距）。两列布局可能导致单列卡片宽度不足。

### 修改策略

1. **增加全局卡片内边距**：将 `.card` 类的内边距从 `p-4` 改为 `p-6`（24px）
2. **调整页面布局宽度**：

- Generate.vue：增加 max-width 限制
- TestCaseGenerate.vue：添加合适的页面容器宽度限制

3. **优化网格布局**：考虑调整两列比例或增大间距
